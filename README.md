# asciitab

Tablature parser and generator.

## Getting started

```bash
# Build the binary from source
go build -o ./asciitab ./main
```

### Create some tabs

```bash
./asciitab random
```
```
Random tab in <B minor [B C# D E F# G A]>
e|--------------|--------------|--------2-----|-----0--0-----|
B|--------------|-----0--2-----|--3--2--------|-----------2--|
G|--------0--4--|--------4-----|-----------0--|--------2--0--|
D|-----------2--|--------------|-----0--------|--4-----------|
A|--4--0--2-----|--2-----------|--------------|--2-----------|
E|--------------|--2-----0--3--|-----------0--|--------------|
```

```bash
./asciitab random --key=C# --scale="major"
```
```
Random tab in <C# major [C# D# F F# G# A# C]>
e|-----------2--|--------2-----|-----2-----4--|--------2--1--|

D|-----1-----1--|-----3--4-----|-----3--------|--4--4--------|
A|--3-----------|-----------4--|--------------|--1-----------|
E|-----1--------|--------------|--------------|--1--4--------|
```
