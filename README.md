# asciitab

Tablature parser and generator.

## Getting started

```bash
# Build the binary from source
go build -o ./asciitab ./main
```

### Create a Random Guitar Tab

```bash
./asciitab random
```
```
Random tab in <G minor [G A A# C D D# F]>
|--------------|--3-----------|-----3--------|-----1--------|
|--------3--1--|-----------4--|--------------|--1--3--------|

|--------------|--------------|--------------|--3--1--1-----|
|--------------|-----0--1-----|--------------|--------------|
|--------------|--1--1--------|--3-----3--1--|-----3--------|
```

```bash
./asciitab random --key=Ab
```
